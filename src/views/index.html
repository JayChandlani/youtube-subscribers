Get Youtube Subscriber

GET /subscribers

Returns an array of all subscribers in the database.

Example:

GET /subscribers

Response:

[ {"_id":"6445244f4cd91e1becbc51d4","name":"Jeread Krus","subscribedChannel":"CNET","subscribedDate":"2023-04-23T12:27:59.407Z","__v":0}, {"_id":"6445244f4cd91e1becbc51d5","name":"John Doe","subscribedChannel":"freeCodeCamp.org","subscribedDate":"2023-04-23T12:27:59.410Z","__v":0}, {"_id":"6445244f4cd91e1becbc51d6","name":"Lucifer","subscribedChannel":"Sentex","subscribedDate":"2023-04-23T12:27:59.411Z","__v":0} ]
GET /subscribers/names

Returns an array of all subscribers' names and subscribed channels in the database.

Example:

GET /subscribers/names

Response:

[ {"name":"Jeread Krus","subscribedChannel":"CNET"}, {"name":"John Doe","subscribedChannel":"freeCodeCamp.org"}, {"name":"Lucifer","subscribedChannel":"Sentex"} ]
GET /subscribers/:id

Returns the subscriber with the specified ID.

Example:

GET /subscribers/64db89eb93f3160b200939c7

Response:

{"_id":"64db89eb93f3160b200939c7","name":"Jeread Krus","subscribedChannel":"CNET","subscribedDate":"2023-04-23T12:27:59.407Z","__v":0}
