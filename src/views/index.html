Get Youtube Subscriber

GET /subscribers

Returns an array of all subscribers in the database.

Example:

GET /subscribers

Response:

[{"_id":"64db89eb93f3160b200939c8","name":"John Doe","subscribedChannel":"freeCodeCamp.org","subscribedDate":"2023-08-15T14:21:31.950Z","__v":0},{"_id":"64db89eb93f3160b200939c7","name":"Jeread Krus","subscribedChannel":"CNET","subscribedDate":"2023-08-15T14:21:31.949Z","__v":0},{"_id":"64db89eb93f3160b200939c9","name":"Lucifer","subscribedChannel":"Sentex","subscribedDate":"2023-08-15T14:21:31.950Z","__v":0}]
GET /subscribers/names

Returns an array of all subscribers' names and subscribed channels in the database.

Example:

GET /subscribers/names

Response:

[{"name":"John Doe","subscribedChannel":"freeCodeCamp.org"},{"name":"Jeread Krus","subscribedChannel":"CNET"},{"name":"Lucifer","subscribedChannel":"Sentex"}]
GET /subscribers/:id

Returns the subscriber with the specified ID.

Example:

GET /subscribers/64db89eb93f3160b200939c7

Response:

{"_id":"64db89eb93f3160b200939c7","name":"Jeread Krus","subscribedChannel":"CNET","subscribedDate":"2023-04-23T12:27:59.407Z","__v":0}
